# agency-instrument-demo

Interactive prototype of the political-agency instrument for the DPI Fund / Groundwork. Twelve core items across four dimensions (personal agency, articulation, participatory efficacy, collective efficacy with horizon) on a 0–4 frequency scale, designed to tell formation apart from mobilization.

Live: https://lizmckenna.github.io/agency-instrument-demo/

Mockup only — not a validated instrument.
